# Linux 基础学习模块

> Linux 是 SRE / DevOps 学习路线的核心基础。

---

## 一、模块目标

本目录用于记录 Linux 学习笔记、动手实验、故障排查过程和实战练习。

目标是从“会使用基础命令”，逐步提升到“能够独立管理和排查 Linux 系统”。
